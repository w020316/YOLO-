## ⚠️ 常见问题

### Q1: 依赖下载太慢？
**A**: 使用 `快速启动.bat`，已配置国内镜像源（阿里云/清华）

### Q2: 端口5000被占用？
**A**: 编辑 `app.py`，修改最后一行端口号：
```python
app.run(..., port=8080)
```

### Q3: 模型加载失败？
**A**: 程序会自动使用预训练后备模型，无需担心

### Q4: 如何训练自己的模型？
```bash
python train.py
```

### Q5: Python版本要求？
**A**: Python 3.7+ (推荐3.8或3.9)
