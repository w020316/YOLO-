系统概述

融合计算机视觉与深度学习技术，为食品加工安全保驾护航

🛡️

智能安全监控

本系统基于 YOLOv8 目标检测算法，针对食品加工场景中的安全规范进行智能化监控。系统能够实时检测食品加工人员是否正确佩戴工作帽，对未佩戴帽子的异常行为进行自动识别与告警。

  • ✓ 实时目标检测，毫秒级响应
  • ✓ 支持图片、视频、摄像头多种输入源
  • ✓ 异常行为自动识别与告警提示
  • ✓ 轻量级 Web 界面，操作简便
  • ✓ 高精度模型，mAP50 达 86.6%

核心功能

多维度检测能力，全方位安全守护

🖼️

图片检测

上传单张图片进行静态检测，快速识别画面中人员的帽子佩戴情况，支持批量处理与结果导出

🎬

视频检测

对视频文件进行逐帧分析，完整追踪检测过程，输出标注后的视频结果与检测统计

📹

摄像头实时检测

接入摄像头进行实时视频流检测，零延迟发现异常行为，适用于生产环境实时监控

🚨

异常告警

检测到未佩戴帽子的人员时自动触发告警，以醒目标注和提示音提醒管理人员及时干预

🟢

hat — 佩戴帽子

NORMAL

快速开始

简单几步，即可部署运行检测系统

1

克隆项目

从 GitHub 克隆项目到本地

git clone https://github.com/your-username/yolov8_project.git
cd yolov8_project
2

安装依赖

安装项目所需的 Python 依赖包

pip install -r requirements.txt
3

启动应用

运行 Flask 服务器，启动 Web 检测界面

python app.py
4

开始检测

浏览器访问 http://localhost:5000，选择检测模式并上传图片或视频即可开始检测
